# Костыли для Ютуба

**КЮ** — маленький userscript для мобильной веб-версии YouTube.

Что делает:
- пропускает спонсорские вставки через SponsorBlock;
- включает полноэкранный режим при повороте смартфона горизонтально;
- рассчитан на Chrome Android + AdGuard.
