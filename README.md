# Костыли для Ютуба

**КЮ** — маленький userscript для мобильной веб-версии YouTube.

Что умеет и это работает:

- SponsorBlock-пропуск работает;
- fullscreen включается по кнопке;
- выход из fullscreen при повороте смартфона вертикально работает;
- свои кнопки '↶10 / ▶ / 10↷' появились и расположены удачно;
- фикс геометрии на xiaomi: две симметричные полосы по краям и полное видео лучше чем обрезанное и с полосой слева.

- скрипт рассчитан на Chrome Android + AdGuard.
